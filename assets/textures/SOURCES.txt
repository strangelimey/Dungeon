Scanned PBR material sources
============================

The following materials were imported from Poly Haven (https://polyhaven.com),
all licensed CC0 (public domain — no attribution required, recorded here for
provenance). Imported via `AssetBaker import` (AO baked into albedo,
displacement packed into the normal map alpha).

  wall_brick      castle_brick_broken_06
  wall_stone      medieval_blocks_03
  wall_moss       mossy_brick
  floor_slabs     seaworn_stone_tiles
  floor_cobble    patterned_cobblestone
  ceiling_rough   rock_face
  ceiling_cracked rock_boulder_cracked

Each set exists at two resolutions, suffixed _1k and _2k; the game picks by
the Settings quality tier (Low/Medium use 1K, High uses 2K). 4K/8K versions
of the same sets are available from Poly Haven under the same asset names.
Everything else in this directory (title_bg.png) is generated by
tools/AssetBaker.
